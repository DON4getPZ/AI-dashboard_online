마케팅 성과 대시보드
네, 맞습니다. 전체/일별/주별/월별 탭을 전환하면 KPI 카드, 차트, 테이블이 모두 해당 집계 단위로 변경됩니다.

예를 들어, "월별" 탭을 선택하면:
• KPI 카드: 월별 집계 데이터의 마지막 월 vs 직전 월 비교
• 차트: 월별 추이 그래프
• 테이블: 월별 상세 데이터
증감율이 "-"로 표시되는 경우는 다음과 같습니다:

• 직전 기간 데이터가 없는 경우: 비교할 기준이 없음
• 직전 기간 값이 0인 경우: 0으로 나눌 수 없음
• 데이터가 1개 기간만 있는 경우: 비교 대상 없음

이 경우 필터 기간을 더 넓게 설정하면 해결될 수 있습니다.
퍼센트 포인트(%p)는 비율의 절대적 차이를 나타냅니다.

예시:
• 지난달 ROAS: 200%
• 이번달 ROAS: 215%
• 변화: +15%p (215 - 200 = 15)

일반 증감율(%)과 다르게, ROAS는 이미 비율(%)이므로 단순 뺄셈으로 변화량을 계산하여 %p로 표시합니다.
CPA(전환당 비용)와 CPC(클릭당 비용)는 낮을수록 효율적인 지표입니다.

따라서:
• CPA/CPC 감소 → 효율 개선 → 녹색
• CPA/CPC 증가 → 효율 악화 → 빨간색

이는 ROAS(높을수록 좋음)와 반대되는 개념입니다.
네, 체크박스를 여러 개 선택하면 됩니다.

• 비용: 왼쪽 Y축 (Bar 차트)
• CPM/CPC/CPA/ROAS: 오른쪽 Y축 (Line 차트)

듀얼 Y축을 사용하므로 단위가 다른 지표도 함께 비교할 수 있습니다. 예: 비용(원) + ROAS(%)를 한 차트에서 확인 가능
필터를 초기화하려면:
1. 각 필터 드롭다운에서 "전체" 옵션을 선택
2. 또는 페이지를 새로고침 (F5)

새로고침 시 데이터가 다시 로드되며 모든 필터가 초기 상태로 돌아갑니다.
현재 버전에서는 직접 내보내기 기능이 제공되지 않습니다.

임시 방법:
1. 테이블 영역을 마우스로 드래그하여 선택
2. Ctrl+C로 복사
3. Excel이나 Google Sheets에 Ctrl+V로 붙여넣기

추후 업데이트에서 CSV/Excel 내보내기 기능이 추가될 예정입니다.
광고 소재별 분석
소재 이미지는 광고 플랫폼(Facebook, Google 등)의 CDN에서 가져옵니다. 다음 경우에 이미지가 표시되지 않을 수 있습니다:

• 광고가 삭제/비활성화된 경우
• 이미지 URL이 만료된 경우
• 네트워크 문제로 로드 실패

이 경우 "이미지 로드 실패" 플레이스홀더가 표시됩니다.
KPI 필터는 최대 3개 조건을 설정할 수 있으며, "또는 (OR)"과 "그리고 (AND)" 연산자로 조합됩니다.

예시: ROAS ≥ 200% 그리고 CPA ≤ 15,000
→ 두 조건을 모두 만족하는 소재만 표시

예시: ROAS ≥ 300% 또는 전환수 ≥ 50
→ 두 조건 중 하나라도 만족하는 소재 표시
ROAS가 0%인 소재는 전환값(매출)이 0원인 경우입니다.

가능한 원인:
• 전환 추적이 제대로 설정되지 않음
• 아직 전환이 발생하지 않음 (러닝 초기)
• 전환값이 GA4/광고 플랫폼에 기록되지 않음

전환 추적 설정을 점검하거나, 충분한 러닝 기간이 지난 후 다시 확인하세요.
A/B 테스트 비교를 위한 추천 워크플로우:

1. 캠페인 또는 광고세트 필터로 테스트 그룹 선택
2. ROAS 내림차순으로 정렬
3. 상위 소재들의 KPI 비교
4. 각 소재 클릭하여 일별 추이 확인

특정 기간의 테스트라면 날짜 필터를 함께 설정하세요.
채널별 분석 & 일반
대시보드 데이터는 매일 자동으로 업데이트됩니다. 각 광고 플랫폼(Meta, Google, Naver, Kakao 등)의 API를 통해 전일 데이터를 수집하며, 보통 오전 중으로 전날까지의 성과가 반영됩니다. 실시간 데이터가 아닌 점 참고해 주세요.
ROAS 100% 미만은 광고비보다 매출이 적다는 의미입니다. 단, 이것이 항상 나쁜 것은 아닙니다. 브랜드 인지도 캠페인이나 신규 고객 확보 목적의 광고는 단기 ROAS보다 LTV(고객 생애 가치)를 고려해야 합니다. 지속적으로 ROAS가 낮다면 타겟팅, 크리에이티브, 랜딩페이지 등을 점검해 보세요.
AI 예측은 과거 데이터 패턴을 학습하여 미래를 예측합니다. 일반적으로 데이터가 많을수록, 패턴이 일정할수록 정확도가 높습니다. 다만, 예상치 못한 외부 요인(시즌, 경쟁사 활동, 시장 변화 등)에 따라 실제 결과는 달라질 수 있습니다. 예측은 의사결정의 참고 자료로 활용하시고, 정기적으로 실제 성과와 비교하여 검증하세요.
네, 대시보드는 반응형 디자인으로 제작되어 모바일 기기에서도 확인할 수 있습니다. 다만, 복잡한 차트와 테이블은 데스크톱 환경에서 보시는 것을 권장드립니다. 주요 KPI와 AI 인사이트 확인은 모바일에서도 편하게 하실 수 있습니다.
각 경고 알림에는 Financial Impact와 Action 가이드가 함께 제공됩니다. Action 가이드에 따라 1차 조치를 취하시고, 추가 분석이나 전략 수정이 필요하시면 담당 Growth Manager와 상의하세요. 경고는 "문제"가 아닌 "개선 기회"로 보시면 됩니다!
현재 대시보드 내 직접 다운로드 기능은 제공되지 않습니다. 데이터 다운로드가 필요하시면 담당 Growth Manager에게 요청해 주세요. CSV 또는 Excel 형식으로 원하시는 기간과 지표의 데이터를 제공해 드립니다.
실무 가이드 & 의사결정
매일 아침 체크해야 할 순서를 알려드립니다:

1단계: AI 인사이트 확인
• 시계열 분석 → 통합 인사이트에서 경고/이상 징후 확인

2단계: KPI 카드 훑어보기
• 마케팅 대시보드 → 전일 대비 ROAS, CPA 변화 확인
• 빨간색 지표가 있으면 즉시 원인 파악

3단계: 문제 채널 드릴다운
• 성과가 급락한 채널/캠페인 클릭하여 상세 분석

소요 시간: 5~10분이면 전체 현황 파악 가능!
예산 배분 결정을 위한 3단계 분석법:

Step 1: BCG 매트릭스 확인
• 퍼널 대시보드 → BCG 매트릭스에서 "Star" 위치 채널 확인
• 트래픽 높음 + 전환율 높음 = 예산 증액 1순위

Step 2: Forecast Matrix 확인
• 시계열 분석 → "Super Star" 세그먼트 확인
• 현재 ROAS 높음 + 미래 성장 예측 = 투자 가치 높음

Step 3: 예산 시뮬레이션
• 예산 슬라이더로 +20%, +50% 시뮬레이션
• 투자 효율이 100% 이상이면 증액 권장
ROAS 하락 시 체계적인 원인 분석 방법:

1. 전체 vs 채널별 비교
• 마케팅 대시보드에서 채널별 ROAS 확인
• 특정 채널만 하락? → 해당 채널 집중 분석
• 전체적으로 하락? → 외부 요인(시즌, 경쟁) 검토

2. 비용 vs 매출 분석
• 비용이 급증했나? → CPC/CPM 상승 여부 확인
• 매출이 급감했나? → 전환율, 객단가 확인

3. 소재별 분석
• 소재별 대시보드에서 ROAS 하락 소재 파악
• 피로도 높은 소재 교체 검토

4. 퍼널 점검
• 퍼널 대시보드에서 이탈 급증 단계 확인
주간 리포트 작성 시 대시보드 활용법:

1. Executive Summary
• 마케팅 대시보드 → "주별" 탭 선택
• 주간 총 비용, 매출, ROAS 요약

2. 채널별 성과 비교
• 채널별 ROAS 테이블 캡처
• Top 3 / Bottom 3 채널 하이라이트

3. 핵심 인사이트
• 시계열 분석 → AI 인사이트 요약 활용
• 주요 이상 징후 및 개선 제안 포함

4. 다음 주 액션 플랜
• Forecast Matrix 기반 투자 방향
• 마이크로 세그먼트 기반 최적화 계획
비전문가에게 설명할 때 유용한 표현:

ROAS 설명
"광고비 100원당 ○○원의 매출이 발생했습니다."
예: ROAS 250% → "광고비 1만원당 2만5천원 매출"

CPA 설명
"한 명의 고객을 확보하는 데 ○○원이 들었습니다."

증감 설명
• 좋은 경우: "지난주 대비 효율이 15% 개선되었습니다."
• 나쁜 경우: "일시적 하락이지만 ○○ 조치로 회복 중입니다."

Tip: 차트 캡처 + 한 문장 요약이 가장 효과적!
광고 ON/OFF 의사결정 기준:

끄는 것을 고려해야 할 때:
• ROAS가 목표 대비 50% 이하로 2주 이상 지속
• 마이크로 세그먼트에서 "Traffic Waste"로 분류
• Forecast Matrix에서 "Problem Child" + 하락 추세

유지해야 할 때:
• 신규 캠페인 (최소 2주 학습 기간 필요)
• 브랜딩 목적 캠페인 (ROAS 외 지표 확인)
• "Rising Star" 또는 "Hidden VIP" 분류

예산만 줄일 때:
• ROAS 목표의 70~100% 수준
• 예산 시뮬레이션에서 "효율 점검" 등급
AARRR 퍼널 대시보드
AARRR은 고객 여정을 5단계로 분석하는 프레임워크입니다:

• Acquisition (유입): 사용자가 처음 방문
• Activation (활동): 첫 의미 있는 행동 (스크롤, 페이지뷰 등)
• Retention (관심): 상품 조회, 장바구니 담기
• Revenue (결제): 결제 시작
• Referral (구매): 구매 완료

각 단계별 전환율과 이탈률을 분석하여 개선 포인트를 찾습니다.
마이크로 세그먼트는 채널별 세부 데이터를 분석하여 4가지 유형으로 분류하고, 각 유형에 맞는 최적화 전략을 제시합니다:

• Hidden VIP: 높은 전환율, 낮은 트래픽 → 예산 확대 권장
• Traffic Waste: 낮은 전환율, 높은 트래픽 → 타겟팅 개선 필요
• Checkout Friction: 장바구니 이탈 높음 → 결제 프로세스 점검
• Rising Star: 성장률 높음 → 적극 투자 권장
각 유형은 다음 조건으로 판별됩니다:

Hidden VIP
• 전환율 ≥ 전체 평균 × 1.5
• 트래픽 ≤ 전체 평균 × 0.5

Traffic Waste
• 전환율 ≤ 전체 평균 × 0.5
• 트래픽 ≥ 전체 평균 × 1.5

Checkout Friction
• 장바구니→결제 전환율 ≤ 30%

Rising Star
• 전주 대비 전환수 성장률 ≥ 50%
• 전환율 ≥ 전체 평균
BCG 매트릭스는 두 가지 축으로 분류합니다:

• X축 (트래픽): 채널별 방문자 수 (전체 평균 대비)
• Y축 (전환율): 채널별 구매 전환율 (전체 평균 대비)

평균 이상이면 "높음", 평균 미만이면 "낮음"으로 분류하여 4분면에 배치합니다.
단계별 이탈 개선 체크리스트:

유입 → 활동 이탈이 높을 때:
• 랜딩 페이지 로딩 속도 체크 (3초 이내가 이상적)
• 광고 메시지와 랜딩 페이지 일관성 확인
• 모바일 최적화 상태 점검

관심 → 결제 이탈이 높을 때:
• 장바구니 리마인더 이메일/푸시 설정
• 할인 쿠폰 팝업 테스트
• 상품 상세 페이지 정보 보강

결제 → 구매 이탈이 높을 때:
• 결제 수단 다양화 (간편결제 추가)
• 결제 프로세스 단계 축소
• 보안 인증 마크 노출
Hidden VIP는 숨겨진 보석! 다음 순서로 활용하세요:

1단계: 원인 분석
• 왜 트래픽이 적은지 파악 (예산 부족? 노출 제한?)
• 현재 타겟팅 설정 확인

2단계: 테스트 확대
• 예산 +30~50% 증액 테스트
• 유사 타겟으로 확장 테스트

3단계: 모니터링
• 2주간 ROAS 변화 추적
• 전환율이 유지되면 추가 증액

주의: 급격한 예산 증가는 체감 수익 효과로 효율 하락 가능!
바로 끄기 전에 개선 가능성을 먼저 점검하세요:

끄기 전 체크리스트:
• 타겟팅이 너무 넓지 않은가? → 세분화 테스트
• 광고 소재가 전환에 최적화되어 있나? → A/B 테스트
• 랜딩 페이지가 해당 채널에 맞는가? → 전용 랜딩 테스트

OFF 결정 기준:
• 2주 이상 최적화 시도 후에도 개선 없음
• ROAS가 목표의 30% 미만 지속
• 예산을 다른 채널에 투입 시 더 높은 ROI 예상

대안: 완전 OFF 대신 예산 80% 감축 후 관찰
시계열 데이터 분석
Forecast Matrix는 현재 ROAS와 예측 성장률을 기준으로 4분면에 배치합니다:

• Super Star (★): 현재 ROAS 높음 + 성장 예측 → 최우선 투자
• Fading Hero: 현재 ROAS 높음 + 하락 예측 → 대체 채널 준비
• Rising Potential: 현재 ROAS 낮음 + 성장 예측 → 테스트 확대
• Problem Child: 현재 ROAS 낮음 + 하락 예측 → 즉시 점검

기준값: ROAS 평균 대비 비교, 예측 성장률 0% 기준으로 상승/하락 판정
예산 시뮬레이션은 체감 수익 모델을 적용하여 계산됩니다:

예산 증가 시:
조정 ROAS = 현재 ROAS × (1 - 0.15 × ln(1 + 예산 변화율))

예산 감소 시:
조정 ROAS = 현재 ROAS × (1 + 0.15 × ln(1 + |예산 변화율|) × 0.5)

추천 등급:
• 투자 효율 ≥ 150%: 증액 추천
• 100% ~ 149%: 유지
• 50% ~ 99%: 효율 점검
• 50% 미만: 감액 검토
광고 예산을 늘릴수록 추가 전환당 비용이 증가하는 현상입니다.

왜 발생하나요?
• 핵심 타겟 소진: 가장 반응 좋은 사용자부터 전환됨
• 광고 피로도: 동일 사용자에게 반복 노출 시 효과 감소
• 경쟁 심화: 예산 증가 시 입찰 경쟁으로 CPC 상승

시뮬레이션 상수: DIMINISHING_FACTOR = 0.15 (15% 체감률)
예: 예산 100% 증가 시 → ROAS 약 10.4% 감소 예상
Prophet 예측의 정확도는 데이터의 양과 패턴의 일관성에 따라 달라집니다. 일반적으로 데이터가 많고, 계절성이 뚜렷하며, 외부 요인 변동이 적을수록 정확도가 높습니다.

광고 데이터의 경우 평균 MAPE(평균 절대 백분율 오차) 10~20% 수준을 기대할 수 있습니다. 예측은 참고용으로 활용하시고, 실제 데이터와 정기적으로 비교하여 검증하세요.
주요 항목 트렌드 탭에서는 세그먼트별로 주요 지표의 시계열 추이를 확인할 수 있습니다:

• 비용 추이: 일/주/월별 광고비 변화
• ROAS 추이: 효율 변화 트렌드
• 전환수 추이: 성과 볼륨 변화
• 매출 추이: 매출액 시계열 분석

각 지표의 트렌드를 분석하여 성과 변화 원인을 파악하고 최적화 전략을 수립하세요.
예산 시뮬레이션을 활용한 최적 증액폭 찾기:

Step 1: 현재 상태 확인
• 시계열 분석 → 예산 시뮬레이션 탭 이동
• 세그먼트 선택 후 현재 ROAS 확인

Step 2: 단계별 시뮬레이션
• 슬라이더로 +20%, +50%, +100% 테스트
• 각 단계별 예상 ROAS와 투자 효율 확인

Step 3: 최적점 찾기
• 투자 효율이 100% 이상 유지되는 최대 증액폭 선택
• 보통 +30~50%가 효율 대비 안전한 증액폭

실무 팁: 한 번에 +100% 이상 증액보다 2주 간격 단계별 증액이 안전!
Fading Hero는 "지금은 좋지만 미래가 불투명"한 채널입니다:

즉시 조치 (1주 이내):
• 해당 채널의 소재 피로도 점검
• 경쟁사 활동 증가 여부 확인
• 새로운 크리에이티브 테스트 시작

단기 대비 (2~4주):
• 대체 가능한 "Rising Potential" 채널 예산 확대
• Fading Hero 채널 예산 점진적 감축 (주 10%씩)

중기 대비:
• 해당 채널 완전 OFF 시 대체할 포트폴리오 구축
• 예측대로 하락 시 빠른 전환 가능하도록 준비

핵심: 당장 끄지 말고, 대안 준비하면서 점진적 축소!
예측 신뢰도 판단 가이드:

예측을 신뢰할 수 있을 때:
• 과거 예측값과 실제값 차이가 20% 이내였음
• 외부 요인(시즌, 프로모션)이 안정적
• 데이터가 6개월 이상 축적됨

예측을 참고만 해야 할 때:
• 최근 예측 오차가 30% 이상
• 새로운 캠페인/채널 (데이터 부족)
• 시즌 변동기 (추석, 연말 등)

차이가 큰 이유 분석:
• 프로모션 일정이 반영 안 됨
• 경쟁사 활동 변화
• 광고 설정 변경 (타겟, 입찰 등)

활용법: 예측값 자체보다 "방향성"을 참고하세요!
트러블슈팅 & 문제해결
로딩 문제 해결 순서:

1. 기본 점검:
• 인터넷 연결 상태 확인
• 브라우저 새로고침 (Ctrl+Shift+R)
• 다른 브라우저로 접속 시도 (Chrome 권장)

2. 캐시 정리:
• Chrome: Ctrl+Shift+Delete → 캐시된 이미지 및 파일 삭제
• 시크릿 모드로 접속 테스트

3. 필터 범위 축소:
• 날짜 범위를 최근 30일로 줄여서 로딩
• 채널/캠페인 필터로 데이터 양 줄이기

계속 문제 시: support@growthmaker.kr로 문의
데이터 불일치 원인과 해결법:

정상적인 경우:
• 광고 플랫폼의 전환 지연 반영 (최대 72시간)
• Attribution Window 내 전환 재집계
• 일별 → 주별/월별 탭 전환 시 집계 방식 차이

확인이 필요한 경우:
• 특정 채널 데이터만 급변 → 해당 플랫폼 점검
• 모든 채널 데이터 급변 → 데이터 파이프라인 확인 필요

조치 방법:
• 광고 플랫폼 관리자 화면에서 교차 검증
• 큰 차이 지속 시 담당 GM에게 연락
데이터 없음 문제 해결:

필터 설정 확인:
• 날짜 범위가 너무 좁지 않은지 확인
• 채널/캠페인 필터가 빈 데이터 조합 아닌지 확인
• "전체"로 필터 초기화 후 다시 시도

데이터 존재 여부:
• 해당 기간에 광고 집행이 있었는지 확인
• 새로 추가된 채널은 데이터 수집에 1~2일 소요

특정 차트만 빈 경우:
• 해당 지표에 데이터가 없을 수 있음
• 예: 전환 추적 미설정 시 ROAS 차트 공백
수치 차이가 발생하는 일반적인 이유:

1. 시간대 차이:
• 대시보드: 한국 시간(KST) 기준
• 광고 플랫폼: 계정 설정 시간대 기준
• → 광고 플랫폼 시간대 설정 확인

2. 데이터 수집 시점:
• 대시보드: 매일 새벽 배치 수집
• 플랫폼: 실시간 업데이트
• → 당일 데이터는 차이 발생 정상

3. Attribution 모델 차이:
• 플랫폼마다 전환 귀속 방식이 다름
• → 대시보드 기준을 표준으로 사용 권장
